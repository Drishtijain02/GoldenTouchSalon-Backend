# Golden Touch Salon Backend рҹ’ҮвҖҚвҷҖпёҸвңЁ

Backend service for Golden Touch Beauty Salon website.

## рҹҡҖ Features
- User authentication (owner login)
- Appointment booking system
- Secure MongoDB database
- Admin dashboard support

## рҹӣ пёҸ Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)

## рҹ”җ Environment Variables
Create a `.env` file with:

JWT_SECRET=your_secret
PORT=5000

## рҹҢҗ API Base URL

https://your-render-url/api
